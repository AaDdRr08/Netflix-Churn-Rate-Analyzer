# Netflix-Churn-Rate-Analyzer
This model will allow you to check if Netflix's recommended TV series, shows, and movies, have a significant effect on their Churn Rate.

Models should be able to run showing you complete results for EDA, Logistic Regression, Decision Tree, and Random Forest, and a SHAP Summary Plot.

Should you replace with a different dataset follow the same format under data dictionary.

Binary classification task was performed with AUC-ROC as primary metric. As per result, their is a clear correlation between Churn Rate and shows recommended to viewers, with Random Forest model showing the best result.
SHAP summary plot shows multiple features order of importance with engagement_intensity being the most significant.
